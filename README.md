# Prathibha Junior And Degree College Website

A professional, responsive website for Prathibha Junior And Degree College located in Draksharamam, East Godavari, Andhra Pradesh.

## Features

- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Multiple Pages**: 
  - Home page with hero section and college overview
  - Courses page with detailed information about all programs
  - Amenities page showcasing facilities
  - Achievements page highlighting milestones
  - Gallery page with filterable image gallery
  - About Us page with mission, vision, and leadership information
- **Modern UI**: Clean, professional design with smooth animations
- **Contact Form**: Interactive contact form for inquiries
- **Google Maps Integration**: Direct link to college location on Google Maps

## Getting Started

1. Open `index.html` in a web browser to view the website
2. All pages are linked through the navigation menu
3. The website uses Font Awesome icons (loaded via CDN)
4. No build process or dependencies required - pure HTML, CSS, and JavaScript

## Customization

### Adding Images
Replace the image placeholders in the gallery and other sections with actual college photos:
- Update the `gallery-image-placeholder` divs in `gallery.html`
- Replace placeholder divs in other pages with `<img>` tags

### Updating Content
- Edit the HTML files directly to update text content
- Modify `styles.css` to change colors, fonts, or layout
- Update contact information in the footer and contact sections

### Color Scheme
The website uses a blue color scheme defined in CSS variables:
- Primary Color: `#1e3a8a`
- Secondary Color: `#3b82f6`
- Accent Color: `#f59e0b`

You can modify these in the `:root` section of `styles.css`.
